# Anvesana-3DPlatformer
**Anvesana** is a 3D platformer set in the sacred landscapes of Nepal. Players take on the role of a guardian navigating culturally inspired environments to restore balance to the Kathmandu Valley.

## 🌄 Concept
The game features four distinct zones inspired by real Nepali landmarks:
- Traditional village courtyard  
- Himalayan snowfields  
- Sacred Buddha head site  
- Swayambhunath Stupa  

Blending cultural heritage with gameplay, Anvesana aims to create a meaningful and immersive experience.

## 📖 Story
An ancient curse has fallen upon the Kathmandu Valley. The great Buddha head lies dormant, its power lost.

To restore balance, the player must collect five sacred **"Om Mani Padme Hum"** prayer flags scattered across the land. Once collected:
- The Buddha head awakens  
- A sacred gate opens  
- The path to Swayambhunath is revealed  

## 🎮 Core Mechanics
- WASD movement (camera-relative)  
- Jump system with ground detection  
- FreeLook camera using Cinemachine (Orbital Follow)  
- Prayer flag collection (trigger-based)  
- Buddha awakening animation (DOTween)  
- Gate opening upon completion  
- Level complete UI (Replay / Main Menu)

## 🛠️ Built With
- Unity (3D)
- Cinemachine
- DOTween

## 🎯 Objective
Collect all sacred prayer flags and restore peace to the valley.
